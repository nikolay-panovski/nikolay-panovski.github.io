Beans' Adventure: Tower of Trickery
Overview

This was part of a project with the assignment of creating a point-and-click game within the Java-based Processing language. The player must first communicate with the puzzle-giving NPCs and uncover the puzzles, then explore other objects in each room to find the solutions. Some puzzles involve cracking a code or solving a riddle by reading the provided clues, others require collecting items in an inventory and using them in the proper place.

v1.0 (Project hand-in)
Tasks during the project:

  • Dialog box system - with flexible conditions to enable (click on story character/ be in a certain room with certain items in inventory) and easy disabling (insert an empty text string in the class)

  • Inventory system - receiving/returning up to 6 collectable objects from/to the current scene

  • Code/sequence puzzle verification system based on integer values

  • Basic menu system
