IMT&S Memory Puzzle with Screens
Overview

This project was part of the ACuTe project for Nova Gorica, for which our team made a city app and conceptualized game ideas.
In the “Memory puzzle with screens” idea, players receive full control over specific displays (via our custom app), but only part of the whole picture at once, which they have to piece together to solve the puzzle. This is inspired by multiple-screen displays and especially suitable for areas with many monitors, like libraries and classrooms.
User control made possible by the Toto platform for city games.

For more detailed information on this prototype and the creation process, please read this blog post.
