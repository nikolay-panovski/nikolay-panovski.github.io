The task during this project was to create a game using innovative technology such as mobile features (e.g. accelerometer, gyroscope), local networking or Augmented Reality. The team set for AR, and we wanted to explore how a Tower Defense game would work in this setting.

v1.0 (Project hand-in)
Tasks during the project:

  • Implement Augmented Reality via Vuforia

  • Recognize specific images on physical cards to spawn buildings, enemies or towers

  • Implement workflow for designers to define properties of enemies and enemy waves easily

  • UI display and handling
