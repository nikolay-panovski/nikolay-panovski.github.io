In this project, I am working in a team for the client WIRWAR, an organization hosting a gaming festival for Twente. Following this theme, the game developed is a competitive multiplayer party game - Bomberman meets Dodgeball in a game where cats and dogs try to hit each other with whatever projectiles they can find, wreak havoc, and score the most points to win!

v0.3 (current after second user playtest)
Tasks during the project:

  • Implement player movement, collisions and different projectile types

  • Implement non-gameplay UI functionality (button selections for scene transitions, characters, levels)

  • Create objects responsible for input / cursors

  • Handle communication between objects with minimum direct coupling (using event queue)

Ongoing tasks:

  • Finish the game loop - implement level selection and winner screens, and carry over player selections to gameplay screens and results out of there

  • Implement timely feedback - particles and animations

  • Remove buggy interactions and find optimization bottlenecks
