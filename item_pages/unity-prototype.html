CityGames
Overview

This concept prototype was made to test multiple ideated games as part of the ACuTe project for Nova Gorica that were made by our team.
Presented are "TV static screen", "Memory puzzle with screens", "Morse code blinker", and "Puzzle pieces compound puzzle". This multifaceted prototype was an extended exercise for me in fast prototype making, forcing me to create a minimal product with consideration only of time (to be able to test and continue with the project), rather than code quality.
List of used assets:

  • First Person Starter Assets by Unity Technologies (including Character controller scripts; all other scripts in this project are mine.)
  • URP Interior House Assets by Unity Technologies
  • Apartment Kit by Brick Project Studio
  • CITY package by 255 pixel studios
  • Polylised - Medieval Desert City by Ink Phantom
  • CubexCube - FREE City Pack I by Cube x Cube
  • Old TV by shadowowy
