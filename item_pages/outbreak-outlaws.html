Outbreak Outlaws
Overview
Team composition:
  • Dmitriy Budnitsky (Artist)
  • Oscar van der Veen (Artist)
  • Quint Winter (Artist)
  • Boaz Theelen (Designer)
  • Maik Rehorst (Designer)
  • Bram Verschuren (Engineer - website)
  • Nikolay Panovski (Engineer - tasks below)

For this project we were assigned to make an arcade game within the custom GXPEngine in C#. We decided on an action platformer where two players fight for biggest progress and highest scores, but cooperatively instead of competitively. This game involves one level with a bunch of puzzles that can only be solved by using unique abilities to help each other.


Originally we had the idea to attempt procedurally generated infinite levels, but at the end we made one long level with a fixed layout using Tiled.

v1.0 (Project hand-in)
Tasks during the project:

  • Gravity and collisions

  • Interactions - with enemies and their bullets, with objects (triggers and pickups)

  • Distance-based melee and ranged enemy attacks

  • UI display and linking with game objects over deaths + scoring (based on distance + enemies + pickups)

  • Basic menu system

  • Randomized spawning of preset enemy groups (idea was scrapped later for balancing reasons)
